## Canciones

### POST /canciones (multipart/form-data)
POST https://subpatronal-heathiest-kash.ngrok-free.dev/canciones

Campos:
- nombre (texto, obligatorio)
- artista (texto, obligatorio)
- album (texto, obligatorio)
- audio (archivo .mp3, obligatorio)
- portada (archivo imagen, obligatorio)

Respuesta (no nulos: id, nombre, artista, album, audioUrl, portadaUrl):
{
  "id": 1,
  "nombre": "Song A",
  "artista": "Artista A",
  "album": "Album A",
  "audioUrl": "/archivos/audio/uuid_song-a.mp3",
  "portadaUrl": "/archivos/portadas/uuid_cover-a.jpg"
}
